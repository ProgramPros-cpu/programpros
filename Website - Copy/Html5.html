Metadata Tags

Metadata tags provide information about the document:

  • <meta> - Defines metadata about an HTML document.
  • <title> - Specifies a title for the document.
  • <base> - Specifies a base URL for relative URLs.
<meta charset="UTF-8">
<title>Page Title</title>
<base href="https://example.com/">

Working Example:

Click to see the example

HTML:

<!DOCTYPE html>
<html>
<head>
<meta charset="UTF-8">
<title>Example</title>
</head>
<body>
<h1>Hello World!</h1>
</body>
</html>

মেটাডেটা ট্যাগগুলি ডকুমেন্ট সম্পর্কে তথ্য প্রদান করে:

  • <meta> - একটি HTML ডকুমেন্ট সম্পর্কে মেটাডেটা সংজ্ঞায়িত করে।
  • <title> - ডকুমেন্টের একটি শিরোনাম নির্ধারণ করে।
  • <base> - আপেক্ষিক ইউআরএলগুলির জন্য একটি বেস ইউআরএল নির্ধারণ করে।

Content Sectioning

Content sectioning tags define the structure of the document:

  • <header> - Defines a header for a document or section.
  • <nav> - Defines a navigation menu.
  • <main> - Specifies the main content of the document.
  • <section> - Defines a section in a document.
  • <article> - Defines an article.
  • <aside> - Defines content aside from the main content.
  • <footer> - Defines a footer for a document or section.
<header>Header content</header>
<nav>Navigation links</nav>
<main>Main content</main>
<section>Section content</section>
<article>Article content</article>
<aside>Aside content</aside>
<footer>Footer content</footer>

Working Example:

Click to see the example

HTML:

<!DOCTYPE html>
<html>
<head>
<title>Content Sectioning Example</title>
</head>
<body>
<header>Header</header>
<nav>Navigation</nav>
<main>Main Content</main>
<section>Section</section>
<article>Article</article>
<aside>Aside</aside>
<footer>Footer</footer>
</body>
</html>

কনটেন্ট সেকশনিং ট্যাগগুলি ডকুমেন্টের কাঠামো সংজ্ঞায়িত করে:

  • <header> - একটি ডকুমেন্ট বা সেকশনের হেডার সংজ্ঞায়িত করে।
  • <nav> - একটি নেভিগেশন মেনু সংজ্ঞায়িত করে।
  • <main> - ডকুমেন্টের মূল কনটেন্ট নির্ধারণ করে।
  • <section> - একটি ডকুমেন্টে একটি সেকশন সংজ্ঞায়িত করে।
  • <article> - একটি আর্টিকেল সংজ্ঞায়িত করে।
  • <aside> - মূল কনটেন্টের পাশাপাশি কনটেন্ট সংজ্ঞায়িত করে।
  • <footer> - একটি ডকুমেন্ট বা সেকশনের ফুটার সংজ্ঞায়িত করে।

Text-Level Semantics

Text-level semantics tags define the structure of text:

  • <h1> to <h6> - Defines HTML headings.
  • <p> - Defines a paragraph.
  • <br> - Inserts a single line break.
  • <hr> - Defines a thematic change in the content.
  • <blockquote> - Defines a block of text that is a quotation.
  • <pre> - Defines preformatted text.
<h1>Heading 1</h1>
<p>This is a paragraph.</p>
<br>
<hr>
<blockquote>Blockquote text</blockquote>
<pre>Preformatted text</pre>

Working Example:

Click to see the example

HTML:

<!DOCTYPE html>
<html>
<head>
<title>Text-Level Semantics Example</title>
</head>
<body>
<h1>Heading 1</h1>
<p>This is a paragraph.</p>
<br>
<hr>
<blockquote>Blockquote text</blockquote>
<pre>Preformatted text</pre>
</body>
</html>

টেক্সট-লেভেল সেমান্টিক্স ট্যাগগুলি টেক্সটের কাঠামো সংজ্ঞায়িত করে:

  • <h1> থেকে <h6> - HTML হেডিংস সংজ্ঞায়িত করে।
  • <p> - একটি প্যারাগ্রাফ সংজ্ঞায়িত করে।
  • <br> - একটি লাইনের ব্রেক বসায়।
  • <hr> - কনটেন্টে একটি থিম্যাটিক পরিবর্তন সংজ্ঞায়িত করে।
  • <blockquote> - একটি উদ্ধৃতি হিসাবে ব্লক অব টেক্সট সংজ্ঞায়িত করে।
  • <pre> - পূর্বনির্ধারিত টেক্সট সংজ্ঞায়িত করে।

Inline Text Semantics

Inline text semantics tags provide meaning to text:

  • <strong> - Defines important text.
  • <em> - Defines emphasized text.
  • <mark> - Defines marked or highlighted text.
  • <small> - Defines smaller text.
  • <del> - Defines deleted text.
  • <ins> - Defines inserted text.
  • <abbr> - Defines an abbreviation or acronym.
<strong>Strong text</strong>
<em>Emphasized text</em>
<mark>Highlighted text</mark>
<small>Smaller text</small>
<del>Deleted text</del>
<ins>Inserted text</ins>
<abbr title="Hypertext Markup Language">HTML</abbr>

  • <strong> - গুরুত্বপূর্ণ টেক্সট সংজ্ঞায়িত করে।

  • <ins> - যোগ করা টেক্সট সংজ্ঞায়িত করে।

Forms

Forms tags are used to collect user input:

  • <form> - Defines an HTML form for user input.
  • <input> - Defines an input control.
  • <textarea> - Defines a multi-line text input control.
  • <button> - Defines a clickable button.
  • <select> - Defines a dropdown list.
  • <option> - Defines options in a dropdown list.
  • <label> - Defines a label for an <input> element.
<form action="/submit" method="post">
<label for="name">Name:</label>
<input type="text" id="name" name="name">
<label for="message">Message:</label>
<textarea id="message" name="message"></textarea>
<button type="submit">Submit</button>
</form>

<button type="submit">Submit</button>

Tables

Tables tags are used to display tabular data:

  • <table> - Defines a table.
  • <tr> - Defines a row in a table.
  • <td> - Defines a cell in a table.
  • <th> - Defines a header cell in a table.
  • <thead> - Groups header content in a table.
  • <tbody> - Groups body content in a table.
  • <tfoot> - Groups footer content in a table.
<table>
<thead>
<tr>
<th>Header 1</th>
<th>Header 2</th>
</tr>
</thead>
<tbody>
<tr>
<td>Cell 1</td>
<td>Cell 2</td>
</tr>
</tbody>
<tfoot>
<tr>
<td colspan="2">Footer</td>
</tr>
</tfoot>
</table>

Working Example:

Click to see the example

HTML:

<!DOCTYPE html>
<html>
<head>
<title>Tables Example</title>
</head>
<body>
<table>
<thead>
<tr>
<th>Header 1</th>
<th>Header 2</th>
</tr>
</thead>
</tbody>
<tfoot>
<tr>
<td colspan="2">Footer</td>
</tr>
</tfoot>
</table>
</body>
</html>

Embedded Content

Embedded content tags include media elements:

  • <img> - Embeds an image.
  • <audio> - Embeds sound content.
  • <video> - Embeds video content.
  • <iframe> - Embeds another HTML page.
<img src="image.jpg" alt="Image description">
<audio controls>
<source src="audio.mp3" type="audio/mpeg">
</audio>
<video controls>
<source src="video.mp4" type="video/mp4">
</video>
<iframe src="https://www.example.com"></iframe>

Click to see the example

HTML:

<!DOCTYPE html>
<html>

এম্বেডেড কন্টেন্ট ট্যাগগুলির মধ্যে মিডিয়া উপাদান অন্তর্ভুক্ত:

  • <img> - একটি চিত্র এম্বেড করে।
  • <audio> - শব্দ কন্টেন্ট এম্বেড করে।

  • <iframe> - অন্য একটি HTML পৃষ্ঠা এম্বেড করে।

Scripting

Scripting tags are used to define scripts:

  • <script> - Defines a client-side script.
  • <noscript> - Defines an alternate content for users that do not support client-side scripts.
<script>
document.write("Hello, world!");
</script>
<noscript>
Your browser does not support JavaScript!
</noscript>

Working Example:

Click to see the example

HTML:

<!DOCTYPE html>
<html>
<head>
<title>Scripting Example</title>
</head>
<body>
<script>
document.write("Hello, world!");
</script>
<noscript>
Your browser does not support JavaScript!
</noscript>
</body>
</html>

স্ক্রিপ্টিং ট্যাগগুলি স্ক্রিপ্ট সংজ্ঞায়িত করতে ব্যবহৃত হয়:

  • <script> - একটি ক্লায়েন্ট-পার্শ্ব স্ক্রিপ্ট সংজ্ঞায়িত করে।
  • <noscript> - যারা ক্লায়েন্ট-পার্শ্ব স্ক্রিপ্ট সমর্থন করে না তাদের জন্য বিকল্প বিষয়বস্তু সংজ্ঞায়িত করে।
